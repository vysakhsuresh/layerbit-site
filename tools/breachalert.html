Zero-Trust K-Anonymity Architecture

How does the Password Scanner work without stealing your password?

  • Local Hashing: When you type a password, Layerbit uses the Web Crypto API to convert it into a SHA-1 cryptographic hash directly inside your browser.
  • Prefix Transmission: We extract only the first 5 characters of that hash and send it to the Have I Been Pwned API. It is mathematically impossible to reverse-engineer your password from just 5 characters of a hash.
  • Client-Side Matching: The API returns a list of hundreds of breached hash suffixes. Layerbit checks that list locally against the remainder of your hash. Your full password never leaves your device.
